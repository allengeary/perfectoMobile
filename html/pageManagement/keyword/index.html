Keyword Driven Testing
com.perfectoMobile.page.keyWord.KeyWordDriver
Keyword-driven testing is a type of functional automation testing framework which is also known as table-driven testing or action word based testing. We provide a simple language defined using XML that allows functional testers and buiness analysts the means of directly created test cases. This eliminates the need for the time consuming two steps process in which testers create manual test cases and then converting them to automated scripts using programming constructs on top frameworks such as UFT or Selenium.

Built on top of the power of Java and Selenium, the Keyword Framework is simple to implement with a low adoption time while being very customizable and extensible. In addition is easo of adoption the framework also added test case unification allowing for the same test case to be used for multiple testing strategies such as Regression, Smoke, User Acceptance and Performance testing using simple tag names.
