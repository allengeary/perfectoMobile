ADD_DEVICE
com.perfectoMobile.page.keyWord.step.spi.KWSAddDevice
Allows you to add multiple devices to a single test case for test chaining across multiple devices. The named device will be added to the running test and available by name using the device attribute on a step. All sub devices will have a separate set of artifacts created. Upon complete of the root device's test case, all devices will be properly shutdown
Attribute Overrides
N/A
Parameters
There are two parameters required for this keyword
1:The name of the device. This is the name used to refer to this device throughout the rest of the test case
1:The device id. This is the ID of the device that you want to acquire from your cloud instance
Examples
This will open a device named secondDevice from the cloud with an ID of 12345
