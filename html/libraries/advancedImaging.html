Advanced Imaging
The Advanced Imaging implementation allows for client (script runner) side implementation of imaging and analysis. By moving some of the imaging functionality to the client side a very rich set of image analysis features can be implemented baed off of OpenCV. It is required the OpenCV be installed on the client machine running the script for the imaging implementation to work correctly
